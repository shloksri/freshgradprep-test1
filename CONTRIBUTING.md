# Contributing Guidelines 🚀

Thank you for your interest in contributing to **FreshGradPrep**.

This project is built through **React Hyderabad Forge**, where contributors collaborate to build real-world products while learning open-source workflows.

---

# Getting Started

1. Fork the repository
2. Clone your fork

> git clone https://github.com/yourusername/freshgradprep.git

3. Create a new branch

> git checkout -b feature/your-feature-name

4. Make your changes

5. Commit your work

> git commit -m "Add: short description of change"

6. Push your branch

> git push origin feature/your-feature-name

7. Open a Pull Request

---

# Contribution Guidelines

Please ensure:

- Code is clean and readable
- Follow the existing project structure
- Write meaningful commit messages
- Avoid unnecessary large changes in a single PR

---

# Communication

Discussions and coordination happen through:

- GitHub Issues
- Forge contributor channels

---

# First Time Contributors

If this is your first open-source contribution — welcome!

Forge is designed to help you understand:

- Git workflows
- Collaboration
- Real product development

Don't hesitate to ask questions.

We are here to learn and build together.
